

# Local Pune Bus (PMPML) Application üöç

A demo mobile application developed as a **Final Year Engineering Project** to improve the local bus transportation experience in Pune city. The application provides real-time bus tracking, route information, and basic passenger features.

---

## üìå Project Overview

The **Local Pune Bus (PMPML) Application** is designed to help commuters track PMPML buses in real time, view routes, and get essential travel information. This project demonstrates the use of **Flutter**, **Node.js**, **Express.js**, **MongoDB**, and **GPS tracking using Raspberry Pi**.
